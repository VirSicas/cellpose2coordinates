# Custom Cellpose Model

This folder contains a **retrained Cellpose model** optimized for **fat body cells of *Drosophila* pupae**.

## Model Details

- **Base model:** Cellpose-SAM  
- **Purpose:** Segmentation of fat body cells for downstream analysis and coordinate extraction.

## Download

- The model is available on Zenodo: [10.5281/zenodo.19158901](https://doi.org/10.5281/zenodo.19158901)
